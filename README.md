# Buscador GitHub

Mini buscador de perfils do GitHub. Feito com React/js.

# Como rodar:

```shel
npm install
npm run start
``` 

# Link para live demo:

[https://stupid-sidewalk.surge.sh/busca](https://stupid-sidewalk.surge.sh/busca)
